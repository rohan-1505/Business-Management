# Business Management System

A Spring Boot based Business Management System developed using Java and Maven.

## Features
- User Management
- Product Management
- Order Management
- Admin Panel

## Tech Stack
- Java 21
- Spring Boot
- Maven
- MySQL
- Hibernate/JPA
- thymleaf

## How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run:
   mvn spring-boot:run
